[xyczero](https://github.com/xyczero ) 的 2014 -> 2015
-------------
### 一. 2014 总结
2014 总体 
2014是快速成长的一年，完成了从校园到职场的蜕变。

#### 成果
1. 自学了Django Web开发框架，简单的Nginx部署，并发布属于自己的博客网站[XycZero's Blog](http://www.xyczero.com )。
2. 开源了一个Android控件[CustomSwipeListView](https://github.com/xyczero/Android-CustomSwipeListView )并得到了[android-open-project](https://github.com/Trinea/android-open-project) 的收录。非常感谢[Trinea](https://github.com/trinea) 的鼓励和支持，坚定了走上开源之路的决心。
3. 经过了初入职场半年多的迷茫，现已明确了自己的职业方向和目标，并坚持不懈为之努力奋斗，绝不妥协。在这里要非常感谢[飞哥](https://github.com/LyndonChin) ，提供了难以衡量的帮助。
4. 慢慢养成了利用业余时间来学习技术的习惯，恩，会努力坚持下去。
5. 加入了[Trinea](https://github.com/trinea) 的Android开源群，找到了很多志同道合的朋友，每天潜着水看他们在群里讨论解答大家的问题，很受感染和鼓舞。

#### 不足
1. 2014前半年没有好好利用属于自己的时间，直至踏入职场后才真正体会到时间的珍贵。
2. 我的[个人博客](http://www.xyczero.com )刚落成不久，虽然更新频率还行，但博客的内容比较单薄。
3. 书读的有些少，无论是专业书籍还是消遣书籍。
4. 虽然Android在业余时间和工作中得到了不少的锻炼，但还是缺乏较为系统全面的学习。

### 二. 2015 展望
2015 总体
唯有努力努力再努力。

1. 多多学习github上的经典开源项目，并争取提交pull Resquest。  
2. 制定一个适合自己的Android学习计划，以一周为单位周期进行全面系统的学习。
3. 多读一些书，恩，至少要把自己在2014年买的书全部读完(::>.<::)。
4. 我的[个人博客](http://www.xyczero.com )不能荒废，要在内容的深度上有所提升。
5. 对于Python，Go语言以及当下比较火的H5还是比较感兴趣的，有机会想深入的了解学习一下。
6. 额，最后一点和大多数程序员一样，要多多努力锻炼自己的身体，因为感觉现在自己的身体的确比在学校时差了许多。
